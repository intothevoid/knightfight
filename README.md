# Knight Fight

A game of Chess written using Pygame. Let the fight begin!

<img src="https://raw.githubusercontent.com/intothevoid/knightfight/main/assets/sshot.png" width="80%" height="80%"></img>

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python3 main.py
```
